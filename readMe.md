

# Persons API Readme


## Getting started

```
git clone <some git hum url>
cd persons-api
npm install

```

## Endpoints

### GET /persons/:id
returns a JSON object representin a person for a given person id
#### Parameters

id - The primary key of the person to retrieve
#### Example call:
GET /persons/person_garver_lois_garber@gmail.com
